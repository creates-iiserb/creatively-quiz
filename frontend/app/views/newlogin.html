Trouble logging in Firefox

If you face trouble logging in, try the steps below

  1. Open a different tab of your Firefox browser and visit about:config
  2. Search network.http.sendOriginHeader and set the value to 1
  3. After setting above value,please refresh this page.
  4. If you have already done the above setting, you can skip these steps.
